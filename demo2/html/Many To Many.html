Many To Many

Many to Many 会在两个 model 中添加一张连接表。

例如，您的应用包含了 user 和 language，且一个 user 可以说多种 language，多个 user 也可以说一种 language。

当使用 GORM 的 AutoMigrate 为 User 创建表时，GORM 会自动创建连接表
